# [Kindling (Grid)](http://timothy-long.com/kindling)

A responsive grid system, with some trace elements of style and markup.

## Usage

### Three Column Layout

```
<div class="container">
		<div class="row flex">
			<div class="span-3 col"></div>
			<div class="span-3 col"></div>
			<div class="span-3 col"></div>
		</div>
	</div>
```
