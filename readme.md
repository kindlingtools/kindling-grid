# [Kindling](http://timothylong.com/kindling)

A pocket-sized grid system; built on the flex display property.

## Basic Usage

**Three-Column Layout**

```
<div class=“row”>
	<div class=“col-4”></div>
	<div class=“col-4”></div>
	<div class=“col-4”></div>
</div>
```

**Offset Column Layout**

```
<div class=“row”>
	<div class=“col-2 offset-6”></div>
	<div class=“col-4”></div>
</div>
```

**Gutters**

```
<div class=“row”>
	<div class=“module”>
		<div class=“col-2 offset-6”></div>
		<div class=“col-4”></div>
	</div>
</div>
```

## Package Management

Bower: ```bower install kindling```
