# [Kindling](http://timothy-long.com/kindling)

A responsive grid system, with trace elements of style and markup.

## Usage

### Three Column Layout

```
<div class="row">
	<div class="span-3"></div>
	<div class="span-3"></div>
	<div class="span-3"></div>
</div>
```

## Package Management

Bower: ```bower install kindling```
