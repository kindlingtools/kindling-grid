# [Kindling](http://timothylong.com/kindling)

A Sass grid system based on the flex display property.

## Usage

**Three-Column Layout**

```
<div class=“row”>
	<div class=“span-4”></div>
	<div class=“span-4”></div>
	<div class=“span-4”></div>
</div>
```

## Package Management

Bower: ```bower install kindling```
