# [Kindling](http://timothylong.com/kindling)

Sass grid framework based on the flex display property.

## Usage

**Basic Three-Column Layout**

```
<div class=“row”>
	<div class=“span-4”></div>
	<div class=“span-4”></div>
	<div class=“span-4”></div>
</div>
```

## Package Management

Bower: ```bower install kindling```
